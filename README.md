# Unlisted

Monorepo for the Unlisted product surfaces.

## Apps

- `apps/mobile` — Expo / React Native app. This is the current primary product surface.
- `apps/web` — Next.js website shell.

## Common Commands

```sh
npm run dev:mobile
npm run dev:mobile:web
npm run build:web
npm run dev:web
npm run build:website
```

Railway currently builds and serves the Expo web export from `apps/mobile`.
