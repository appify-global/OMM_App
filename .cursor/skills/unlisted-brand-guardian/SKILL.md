# Unlisted Brand Guardian

## Brand Position

Unlisted is a buy-only private property discovery platform built to challenge Domain and REA with a calmer, more premium experience.

The product should feel:
- Professional, restrained, credible
- Search-first and utility-led
- Premium without decorative luxury cliches
- Minimal, not empty
- Buyer-focused, not seller/agent-first

## Non-Negotiables

- Users can only buy/search properties on Unlisted.
- Do not introduce rent, sold, commercial, broad marketplace, or seller-first flows unless explicitly requested.
- Do not make UI feel like a generic SaaS landing page.
- Avoid oversized buttons, loud cards, heavy shadows, excessive gradients, and inflated spacing.
- Avoid bold text except for hero moments, key prices, or primary calls to action.
- Prefer product credibility over marketing hype.

## Core Palette

Use the mobile app palette as the source of truth:
- Forest: `#0f2918`
- Forest pressed: `#0a1f12`
- Charcoal: `#1a1a1a`
- Text: `#111111`
- Muted text: `#6B6B6B`
- White: `#ffffff`
- Cream/input surface: `#f5f1ed`, `#f7f3f0`
- Sage: `#8a9b8e`
- Borders: `#E5E5EA`, `#E0E0E0`

## Typography

- Use medium and regular weights for most interface text.
- Reserve heavy/bold styling for hero headlines and rare emphasis.
- Prefer clean sans-serif for product UI.
- Use display serif sparingly for premium editorial moments.

## Decision Filter

Before finalizing a change, ask:
1. Would this feel credible beside Domain or REA?
2. Is the UI helping buyers search faster?
3. Did we avoid unnecessary visual noise?
4. Does this match the mobile app's palette and restraint?
5. Is anything oversized just because it looks "designed"?
