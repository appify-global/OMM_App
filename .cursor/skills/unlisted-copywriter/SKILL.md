# Unlisted Copywriter

## Voice

Unlisted copy is confident, direct, premium, and practical. It should sound like a serious property product, not a hype-heavy startup.

## Audience

Primary audience:
- Buyers looking for private property opportunities
- High-intent movers
- Premium property seekers

Secondary audience:
- Agents responding to qualified buyer demand

## Copy Rules

- Keep copy concise.
- Prefer utility over hype.
- Use plain property language.
- Avoid buzzwords like "revolutionary", "seamless", "unlock your journey", and "game-changing".
- Avoid overexplaining the concept of off-market property.
- Make CTAs specific and low-friction.

## Preferred Terms

Use:
- Private listings
- Buyer briefs
- Saved searches
- Request access
- View map
- Create a brief
- Search properties
- Message agent
- Private campaign

Avoid:
- Rent
- Sold
- Commercial
- Marketplace for everything
- Seller-first framing
- Loud SaaS phrases

## CTA Guidance

Good:
- Search
- View map
- Save search
- Request access
- Create a brief
- Get the app

Bad:
- Start your journey
- Unlock opportunities now
- Discover endless possibilities
- Revolutionize your property search
